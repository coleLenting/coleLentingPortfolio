React Project - FitJourney

FitJourney is a React-based application designed to help users track their fitness goals and progress. The project involved creating a dynamic, responsive interface that allows users to add, update, and visualize their fitness goals. The app leverages React's component-based architecture to ensure scalability and maintainability. Key features include goal tracking, progress visualization, and a user-friendly design optimized for both desktop and mobile devices.
